Overview

Utilities to convert dates to and from unix time.

Supported in:
  • .NET Framework 3.5
  • .NET Framework 4.0
  • .NET Framework 4.5
  • Silverlight 5
  • .NET for Windows Store apps
  • Windows Phone 7
  • Windows Phone 8
